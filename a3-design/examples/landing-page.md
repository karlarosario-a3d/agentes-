# Example: Landing Page

A textual reference for an institutional/product landing page using A3Data's brand colors. The manual targets DataViz, so web type scale and components are `needs-review`.

## Intent

A marketing/product page that communicates A3Data's data expertise with a clean, professional look.

## Theme options

- **Light theme:** background `#F5F5F5`, text `#00083D`.
- **Dark theme:** background `#001863` (Deep Sea), text `#F5F7FF`.

## Hero

- **Background:** dark `#001863` for impact (or light `#F5F5F5`).
- **Logo:** dark-background lockup on dark hero (original file only).
- **Headline:** Segoe UI, Bold — size `needs-review` (no web H1 scale defined); color `#F5F7FF` on dark.
- **Subhead / body:** Segoe UI, Normal — size `needs-review`; color `#F5F7FF` on dark.
- **Accent:** Pink Nic `#F53199` or Tech Blue `#0033FF` for a highlighted word or CTA emphasis.
- **CTA button:** `needs-review` (no button component) — use Tech Blue `#0033FF` with accessible contrast and base-8 padding, flagged.

## Sections

- **Spacing:** base-8 — section padding in multiples of 8 (e.g., `24px`+).
- **Surfaces/cards:** `#FFFFFF` on light, with subtle divider `#EBEBEB` (inferred).
- **Accent palette for highlights/illustrations:** secondary colors Summer Time `#FFEE00`, Amethyst Velvet `#7B4AAF`, Sky Frost `#39D0FF`.

## Data showcase

- If embedding charts, follow `docs/data-visualization-guidelines.md` (categorical palette in order, DIN for numbers).

## Iconography

- Phosphor Icons 2.1, Stroke / Regular, single consistent weight.

## Footer

- Dark `#001863` background, text `#F5F7FF`.
- Brand attribution per A3Data guidelines.

## Accessibility notes

- Verify hero text contrast (`#F5F7FF` on `#001863` is strong).
- CTAs need visible focus and text labels.
- Don't rely on accent color alone for meaning.

## Needs-review for this screen

Web type scale (H1–H6, body, caption), button/CTA specs, radius/shadows, responsive breakpoints, container widths, section rhythm beyond base-8.
